# Hướng dẫn chạy code:
    B1: Cài đặt XAMPP.
    B2: Tải code và lưu vào folder htdocs trong folder xampp.
    B3: Chạy file xampp-control trong folder xampp và start Apache và MySQL.
    B4: mở trình duyệt và chạy link localhost/read_stories_web.
